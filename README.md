<h1 align="center">Assignment Static Portfolio Site</h1>


## How to Run

Clone The Repository:
```
git clone  https://github.com/jahid32/protfolio.git 
```
Install composer dependency:
```
composer install
```

The run following command to run local server :
```
php artisan serve
```
